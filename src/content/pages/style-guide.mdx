## Typography

The site uses the system font stack via Tailwind CSS defaults. No custom web fonts are loaded. Prose styling is provided by `@tailwindcss/typography`.

### Headings

### Body Text

Regular paragraph text demonstrates the base body style. **Bold text** and _italic text_ are styled by the typography plugin. [Inline links](#) use the default prose link style with underline.

> Blockquotes appear like this. They can be used for callouts, quotes, or emphasis.

### TextEffect

Animated text reveal component using `motion/react`. Requires a `client:` directive for hydration.

**Presets:** `blur`, `fade-in-blur`, `scale`, `fade`, `slide`

**Segmentation:** `per="word"` (default), `per="char"`, `per="line"`

| Prop           | Type         | Default  | Description                             |
| -------------- | ------------ | -------- | --------------------------------------- |
| `text`         | `string`     | —        | Text content to animate                 |
| `preset`       | `PresetType` | `"fade"` | Animation preset                        |
| `per`          | `PerType`    | `"word"` | Split text by word, character, or line  |
| `as`           | `string`     | `"p"`    | HTML element to render as               |
| `delay`        | `number`     | `0`      | Delay before animation starts (seconds) |
| `speedReveal`  | `number`     | `1`      | Stagger speed multiplier                |
| `speedSegment` | `number`     | `1`      | Individual segment animation speed      |
| `trigger`      | `boolean`    | `true`   | Whether to trigger the animation        |

```astro
import {TextEffect} from '@/components/ui/text-effect';

<TextEffect
  client:load
  preset="fade-in-blur"
  per="word"
  speedReveal={1.1}
  speedSegment={0.75}
  delay={0.25}
  as="h1"
  className="text-4xl tracking-tight"
  text="Your heading text here"
/>
```
